# Product Photos

Google Driveの商品写真をダウンロードし、このフォルダに置きます。

`content/story_schedule.csv` の `source_image` に合わせて、まずは以下の名前にするとそのまま使えます。

- `tablet.jpg`
- `ingredients.jpg`
- `gift.jpg`
- `cocoa.jpg`
- `flavors.jpg`
- `nama.jpg`
- `family.jpg`
- `egift.jpg`
- `donation.jpg`

別の名前にする場合は、CSVの `source_image` を変更してください。
